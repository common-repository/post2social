== Description ==

[Documentation](https://codeweby.com/documentation/post2social/ 'Documentation') | [Support](https://codeweby.com/contact-us/ 'contact us')

Post2Social is an easy to use All-In-One Twitter social plugin included with Twitter Cards and Twitter Auto Post.

**Bring more people from Twitter to your website**

Twitter Cards is a powerful ad format that allows users to preview an image, related context, and a clear call to action in their timeline.

This perfect Twitter all-in-one social plugin will help you to keep your visitors flow high.

**Use Twitter Cards to:**

* Grow your site traffic.
* Increase online purchases.
* Drive specific actions.

**Twitter Cards**

Easily add Twitter Cards Meta into all of your WordPress pages, posts, categories, taxonomies, **custom post types**, search results and homepage.

**Included with Summary Cards and Summary Cards with Large Image**

**Guaranteed Twitter Cards validation and approval within a few seconds!**

**Features**

* **Easy to use and clean admin interface.**
* Multisite compatible.
* Enable/Disable Twitter Cards by a single click.
* Included with Summary Cards and **Summary Cards with Large Image**.
* Ability to assign the card image from a **custom field, use the first image from the content** or use the posts/pages **featured image**.
* Automatically add Twitter Cards Meta into every of your WordPress **pages, posts and custom post types**.
* Possibility to add Twitter Cards Meta to your Homepage using the default image and description.
* **Card Type can be defined individually for every single post, pages and even for custom post types.**
* Include/ Exclude each of the Twitter Card Meta tags e.g. (Title, Description, URL, Image, Author, Publisher)
* Display the latest tweets widget. 

**Twitter Auto Post**

With Twitter Auto Post you can **easily auto post your posts, pages and custom post types to Twitter** soon as you hit the publish button. **You can define individually the Twitter Card Type you want to use for every single pages, posts and custom post types.**

Once you have enabled the auto post feature and selected a Twitter Card Type anytime you hit the publish button your content will be auto posted to Twitter. The system will “remember” for every single pages setting so you don’t need to define again once you open the page.

**Features**

* Easy to use and clean admin interface.
* **Enable/Disable Twitter Auto Post by a single click.**
* Multisite compatible.
* Configure your Twitter account which you want to use for auto posting.
* **Twitter Auto Post and Twitter Card Type can be defined individually for every single post, pages and even for custom post types.**
* **Define individually the post types you want to display on the “Share on Social Metabox”** e.g. (posts, pages and custom post types)
* Enable/Disable Auto Post on every single pages, posts and custom post types.
* **Auto Convert the added words into hashtags.** e.g. (WordPress will become to # WordPress)
* **Manually post a Tweet form your WordPress admin.**
* Display the latest tweets widget.
* URL shortens by default via Twitter. 

**Post2Social Auto Publisher**

What is Auto Publisher?

Auto Publisher is an add-on for the Post2Social WordPress plugin.

Auto Publisher allows you to auto publish any of your WordPress website post, pages and custom posts on the selected social networks based on the defined interval.

== Installation ==

Installing Post2Social is easy! Please follow the above steps.

= Using The WordPress Dashboard =
1. Navigate to the 'Add New' in the plugins dashboard.
1. Search for 'Post2Social' look for the author 'Codeweby'.
1. Click 'Install Now'.
1. Activate the plugin on the Plugin dashboard.
1. A new menu item 'Post2Social' will appear in the main menu.

= Uploading in WordPress Dashboard =
1. Navigate to the 'Add New' in the plugins dashboard.
1. Navigate to the 'Upload' area.
1. Select cataloggi.zip from your computer.
1. Click 'Install Now'.
1. Activate the plugin in the Plugin dashboard.
1. A new menu item 'Post2Social' will appear in the main menu.

= Using FTP =
1. Download post2social.zip.
1. Extract the cataloggi directory to your computer.
1. Upload the cataloggi to the /wp-content/plugins/ directory.
1. Activate the plugin in the Plugin's dashboard.
1. A new menu item 'Post2Social' will appear in the main menu.
